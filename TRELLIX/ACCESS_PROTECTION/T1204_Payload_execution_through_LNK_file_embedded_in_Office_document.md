## Author
Trellix

## Description
This rule trigger indicates an attempt to create a .lnk file from a Word application.   

## Rule Class 
Files

## Rule TCL
```tcl
Rule {
  Process {
      Include OBJECT_NAME { -v "winword.exe" }
  }
  Target {
    Match FILE {
      Include OBJECT_NAME { -v "**\\temp\\*.lnk" }
      Include -access "CREATE"
    }
  }
}
```
