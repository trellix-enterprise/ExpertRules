## Author

## Description
This rule trigger indicates an attempt to abuse PSExec by using named pipes to transfer standard output, input and error.  

## Rule Class 
Files

## Rule TCL
```tcl
Rule {
    Process {
        Include OBJECT_NAME {
            -v *.exe
        }
    }
    Target {
        Match FILE {
            Include OBJECT_NAME {
                -v "**pipe\\psexesvc*"
            }
            Include OBJECT_NAME {
                -v "**pipe\\remcom*"
            }
            Include OBJECT_NAME {
                -v "**pipe\\PAExec*"
            }
            Include OBJECT_NAME {
                -v "**pipe\\csexec*"
            }
            Include -access "CONNECT_NAMED_PIPE"
        }
    }
}
```
